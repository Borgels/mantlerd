# Contributing to mantlerd

Thanks for your interest in improving `mantlerd`.

## Development setup

1. Fork and clone the repository.
2. Ensure you have Go installed (see `go.mod` for the required version).
3. Run tests:

```bash
go test ./...
```

## Before opening a pull request

- Run the full test suite:

```bash
go test ./...
```

- Keep PRs focused and include context in the description.
- Add or update tests when behavior changes.

## Commit and PR guidance

- Use clear commit messages that explain intent.
- Link related issues in PR descriptions when applicable.
- Include relevant logs/output for behavior or protocol changes.

## Code of conduct

By participating, you agree to follow the project [Code of Conduct](./CODE_OF_CONDUCT.md).
